Операционные системы
  • Windows
    Грешно не знать самую популярную систему на персональных компьютерах.
  • Windows Server
    В некоторых случаях использовал эту систему для стриминга и записи аудио-потоков для радио-эфиров.
  • Fedora
    Использую систему на пользовательских ПК и серверах. Да, на некоторых серверах. 😄
  • CentOS / Oracle Linux
    Использую эту промышленную систему исключительно на серверах.
  • Ubuntu / Debian
    Ubuntu использовал в качестве домашней системы. На Debian работает несколько серверов в организациях.
  • openSUSE
    Неплохая система с автоматическим и простым конфигуратором. Раньше использовал в качестве домашней системы.
  • Proxmox
    Использую Proxmox как платформу для развёртывания виртуальных машин.
  • OPNsense
    Работает в качестве шлюза и NAT между локальной сетью и внешним интернетом.
Приложения

Устанавливаю и настраиваю компоненты системы, например, такие, как:

  • Apache HTTPD
    Сервер для обработки веб-запросов.
  • NGINX
    Использую как proxy-сервер к Apache HTTPD, так и отдельно для отдачи контента.
  • Elasticsearch
    Работал в качестве сервера поиска. Настраивал локальную node.
  • Memcached
    Кэширующий сервер для хранения данных в RAM. Использовал его для снижения нагрузки на веб-сервера.
  • Redis
    После Memcached, перешёл на Redis.
  • MongoDB
    Использовал эту базу данных для работы с приложениями, написанными на NodeJS.
  • MySQL / MariaDB
    Стандарт для веб-приложений. Использовал как MySQL, так и MariaDB, что по сути, одно и тоже. Только в последних версиях они начали чуть отдаляться друг от друга.
  • VsFTPD
    Очень безопасный FTP-сервер.
  • Icecast
    Когда то поднимал трансляции радио-эфиров в эти ваши интернеты.
  • …и многие другие (чем я только не занимался…).
CMS / CMF
  • Drupal
    Предоставляет очень мощные возможности по управлению контентом. Люблю эту CMF.
  • Flarum
    Неплохой, развивающийся движок форума. Являюсь разработчиком и куратором русской локализации этого движка.
  • MediaWiki
    Для составления документации нет лучше инструмента, чем WIKI-система. Умею поднимать с единой системой авторизации пользователей и единым хранилищем медиа. Настраиваю WIKI-фермы.
  • WordPress
    Очень простой движок для сайтов различного назначения. Делаю небольшие модули и темы для этого движка.
  • XenForo
    Мощный движок для создания форума. От разработчиков vBulletin. Делаю небольшие модули и темы для этого движка.
